# farm-iot-api
An API between Farm management platforms and IoT platforms

This API came out of a need for IoT platforms used in a low-resource farm context to integrate with farm management platforms. 

The challenge was to avoid a situation where every pair (IoT <> Farm) had to be separately implemented, which would probably not happen, creating lock-in for users of farm platforms to use particular IoT devices, or for owners of particular IoT devices to use a specific platform.

See the original [grant proposal](A%20shared%20API%20between%20IoT%20devices%20and%20Agroecology%20Platforms.pdf)for more information on the goals.

Our first step was a [Needs Assessment](Farm%20IoT%20API%20-%20State%201%20-%20Needs%20Assessment%20-%20Google%20Docs.pdf)

We are now at the stage of an [initial draft API](API.md)

Current collaborators are: 
* [Frugal IoT](https;//frugaliot.naturalinnovation.org) - Mitra Ardron (lead editor)
* [Lite Farm](https://www.litefarm.org/) - Divya Chayanam
* [OurSci](https://www.our-sci.net/) / [SurveyStack](https://app.surveystack.io/) - Greg Austic & Dan TerAvest

We welcome more collaborates, please get in touch, and raise issues if the API does not meet the needs of your platform.

This is still a draft, there is a partial implementation in Frugal IoT, so until there are at least two partners interchanging data this is likely to change. 

Please feel free to open an issue if you notice any problems./
